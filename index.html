Onde consigo a chave de API? É seguro?

Gere uma chave em console.anthropic.com → API Keys. A chave fica salva apenas neste navegador (localStorage) e é enviada direto à API da Anthropic — este arquivo não tem servidor. O uso da API tem custo por documento processado.
